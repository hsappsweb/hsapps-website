A modern approach to mass animal care

 
 
 

Technology that streamlines field operations and improves animal welfare

Join Now

Free for qualified organizations

 
 
 

The Mass Vaccination app streamlines administering large number of vaccinations in the field to ensure resources are optimally deployed and the maximum number of animals impacted.

Key Features

  • Mobile (Android) application for data collection: GPS tracking, barcode scanning and image capture with off-line support/sync

  • Web application for administration: project setup, monitoring and data analysis

  • Geofenced areas to separate teams into distinct work areas for maximized efficiency

  • Survey feature for evaluating vaccination coverage

    • Learn more
     
     

    Key Features

    Records animals precise home location (GPS) for later return

    Release map ensures optimal route planning and GPS lock requires animal released within 15M of home

    Surgery data gathering: record spay/neuter details with RFID chip barcode scanning

    Web app for administrators to setup and monitor project status and analyze data
