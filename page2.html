Streamline population management efforts by optimizing workflow throughout the catch, neuter & release process. Field users gather field & clinical data for program monitoring using a mobile app while administrators setup projects and analyze results through a web app.

Field Data Gathering with Offline Sync

Record animal picture, GPS location, and other information about each animal captured.

Offline/Sync feature enables smooth handling of connectivity issues.

Designed for field work, the app has been designed to work with poor internet connectivity — data is cached to sync later when better connectivity is restored. This feature ensures tamper-proof data and improves standardization.

Release Map & GPS “lock”

Optimally route plan releasing animals by mapping all target release destinations
The GPS lock feature requires the field team to release an animal within 15M of where it was picked up ensuring the animal isn’t stressed by having to find a new home and pack

Clinic records management

From capture to clinic care to release, a built-in health records system captures all the details of the sterilization procedure as well as exams or other treatments while the animal is in-care at the clinic

A sister phone app facilitates RFID barcode scanning.  Both the web app and phone app are designed to handle spotty internet connections caching data until connections are restored.

Reporting & Data Export

The HSIApps Web application provides a dynamic reporting system with options to view pre-defined reports and download data into CSV for custom analysis.
